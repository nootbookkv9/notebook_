# Infix and Postfix and Prefix ###############################

## Infix expression
  --> x + y
## Postfix expression
  --> x y +
## Prefix expression
  --> + x y

# Postfix and Prefix VS Infix
  - no need to parentheses to determine operations precedence
    - (x + y) * z --> x y + z *
  - That makes them easier to computer to evaluate these expression
