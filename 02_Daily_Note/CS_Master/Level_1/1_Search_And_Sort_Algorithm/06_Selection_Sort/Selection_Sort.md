# Selection Sort #############################################
## Psuedo Code
for i = 0 to n
  find the smallest element from i to n
  swap whth the i element

## Analysis
  [ Worst Case ]
    O(n^2)
  [ Best Case ]
    O(n^2)

## Time Complexity --> O(n^2)

## Space Complexity --> O(1)

## In-place

## Stable
