## Exponentials & Logarighms ===============================

# Power properties

- a**0 = 1
- a**-n = 1/a**n

- a**n * a**m = a**(n+m)
- a**n / a**m = a**(n-m)

- a**n * b**n = (a*b)**n
- a**n / b**n = (a/b)**n

- (a**n)**m = (a)**n*m

# Logarithms
- Invers of power

- log a x = n <--> a**n = x
- log a x --> x/a/a/a --> 1 --> n

- log x = log 10 x
- lg x  = log  2 x

  # Logarithms properties

  - log (x*y) = log x + log y
  - log (x/y) = log x - log y
  
  - log (x**y) = y * log x
